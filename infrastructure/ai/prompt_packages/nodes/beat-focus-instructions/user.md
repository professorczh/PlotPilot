【格式与段落法则】
1. 段落是意义的集合，不是句子的分行。属于同一视觉焦点、同一动作链或同一心理转折的句子，必须合并在同一段落里，用句号衔接而非换行切分。
   ❌ 错误示范："伊莎贝拉走到马车旁边。她停住了。看着那三张脸。都很年轻。二十多岁。"（每个短句强制独立，像分镜脚本）
   ✅ 正确示范："伊莎贝拉在马车旁停下脚步。三具年轻的尸体静静躺在车里，脸上冰霜尚未融化，瞳孔里凝固着极度的恐惧——都不过二十多岁。"
2. 段落长度应有呼吸感：铺陈/叙事段落2-5句（约80-200字），高潮暴击可用1句独段（仅限冲突引爆/重大揭露/情绪转折的瞬间），但全章独段比例不得超过15%。
3. 对话可以独立成段，但连续短对话（3句以内的一问一答）应合并为一段对话流，避免对话变成打字机式的一行一句。
4. 场景转换时空一行分隔；同一场景内的动作、观察和心理应聚合在同一段落里。
5. 适配手机阅读：段落不宜过长（不超过250字），但绝不要为"留白"而把有机整体拆成碎片——留白来自段落间的空行，不是来自段落内部的强制换行。

⚠️ 篇幅控制与收尾规则（必须遵守）：
- 剧情已完毕立即收束，不拖沓
- 禁止无意义的对话重复或空洞感叹凑字数
- 结尾必须完整（以句号、问号或叹号结束），绝不允许中途截断
- 最后节拍收尾用画面/动作/对话，禁止抽象总结

📝 章节叙事节奏（专业小说家法则）：
- 第一节拍（起）：用画面抓住读者——一个动作、一句话、一声巨响
- 中间节拍（承/转）：冲突层层升级，每段都有信息增量
- 最后节拍（合+钩）：阶段结果+悬念钩子——一个画面、一句话或一个未解的动作
- 节拍之间自然衔接，像长镜头一镜到底
